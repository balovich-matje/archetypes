# Classes (working title)

An RPG class mod for Minecraft Fabric — pick your class in the first minute, from the
inventory screen. No crafting station, no spell book, no weapon coupling.

Early planning. Nothing implemented yet.

- [notes/design.md](notes/design.md) — what we're building and the open questions
- [notes/rpg-series-reference.md](notes/rpg-series-reference.md) — what the RPG Series
  does, for reference

Sibling project: [Specialities](https://github.com/balovich-matje/specialities) — the
skills mod this is designed to sit alongside.
